Eskimo-1.0

Eskimo is a Flex mobile library providing native control patterns to your apps on multiple mobile.

To use it:
  * Add the eskimo.swc in the library path of your project.
  * Import the eskimoTheme.swc in the Flex theme of your project.
  * Run and look the sources of "tour d'Eskimo" project available in the sample directory.
  * Check the asDoc.
